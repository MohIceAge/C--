# Compilateur C-- (fait dans le cadre d'un projet de l'ENS Paris-Saclay)
Le langage C-- est le langage C avec quelques fonctionnalités en moins.
Le dossier c--.1 contient le compilateur et quelques examples de programmes C--.
Le dossier fling contient un petit jeu créé pour se familiariser avec le langage Ocaml.
